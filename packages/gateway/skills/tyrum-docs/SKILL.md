Use this skill when the user asks how Tyrum works, is configured, or is supposed to be used.

- Prefer `https://docs.tyrum.ai` for public product and operator guidance.
- Prefer the local `docs/` tree when the question is about the current checkout's implementation or architecture.
- Treat public docs as usage guidance, not as authority over the current code when the two differ.
- Do not browse the docs on unrelated turns.
