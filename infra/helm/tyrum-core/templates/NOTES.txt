This chart deploys the Tyrum control-plane stack (policy, api, web, mock-llm)
plus the dev dependencies (Postgres + Redis). After the release becomes ready,
run:

  helm test {{ .Release.Name }}

to execute the readiness probes against each service endpoint.

Services ship with sensible defaults for ConfigMaps, Secrets, and HPAs. See the
README entry in docs/infra/helm.md for override examples.
